# 프로젝트 개요

**어디더라**는 인스타그램 릴스를 통해 발견한 장소를 저장하고, 이를 기반으로 `사용자 조건 기반 데이트 코스 자동 생성`을 지원하는 서비스입니다.

기존에는 릴스를 보며 마음에 드는 장소를 발견해도 별도로 기록하거나 다시 찾기 어려웠고, 저장하더라도 지도, 메모, SNS 등 여러 도구에 분산되어 관리되면서 
사용자가 원하는 흐름에 맞춰 코스를 구성하는 데 추가적인 시간과 노력이 필요했습니다.

어디더라는 릴스 공유 기능을 통해 장소를 서비스 내부에 구조화된 데이터로 저장하고, 사용자가 선택한 지역과 카테고리, 그리고 원하는 순서를 기반으로 
조건에 맞는 장소들을 조합하여 코스를 자동으로 생성합니다.

이를 통해 단순 저장을 넘어 `발견 → 저장 → 조건 설정 → 코스 생성`까지 이어지는 흐름을 지원하도록 설계했습니다.
