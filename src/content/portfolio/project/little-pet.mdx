# 프로젝트 개요

**리틀펫**은 소동물 반려인을 위한 라이프 케어 플랫폼입니다.
UMC 7기 프로젝트로 진행되었으며, 소동물 반려인들의 일상적인 케어와 정보 공유를 돕는 서비스를 제공합니다.
사용자 중심의 UI/UX 디자인을 통해 반려인들이 쉽고 편리하게 사용할 수 있도록 설계했습니다.
